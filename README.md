## 机房IP和住宅IP，Facebook看起来有多不一样？

先说一个基本概念，方便后面对比方案用。

**美国住宅IP**，是指由美国本地ISP（如AT&T、Comcast、Verizon）分配给真实家庭用户的公网IP地址。这类IP在各大平台的IP数据库里，归属显示为家庭宽带用户，欺诈风险评分极低，信任度高。

机房IP的问题是，它不管伪装得多好，在IP质量数据库里的归属还是数据中心。Facebook的风控系统每天处理数十亿次登录请求，识别机房IP是基本操作。从国内代理、VPN连Facebook广告后台，平台会记录：这个IP属于某个数据中心的某个ASN，上面跑了成百上千个账号，这是典型的批量操作特征。

触发判断之后，后续的事就很熟悉了——要验证身份证，要刷脸，要提交营业执照，然后还是封。

住宅IP则完全不同。一个AT&T家庭宽带分配的IP，在平台眼里就是一个普通美国人的家庭网络。这个IP上可能挂着一两个Facebook账号、一个Gmail、偶尔刷刷TikTok，和真实用户的行为模式高度一致。风控触发的概率自然低得多。
